# -Low-Birth-Weight-LBW-Prediction-Using-Machine-Learning
Predict Low Birth Weight (LBW) using ML with preprocessing, feature selection, hyperparameter-tuned Decision Tree, Random Forest, Logistic Regression, SVM, XGBoost. Evaluated with Accuracy, Kappa, Recall, Specificity, F1, G-Mean, AUC-ROC. Includes feature importance, SHAP/Waterfall plots, ROC &amp; PR curves.
