## ◈ EVALUATION METRICS

<div align="center">

```
 METRIC          FORMULA                                        BEST FOR
 ─────────────────────────────────────────────────────────────────────────
 Accuracy      = (TP+TN) / (TP+TN+FP+FN)               Balanced datasets
 Kappa         = (Po - Pe) / (1 - Pe)                   Imbalanced classes
 Sensitivity   = TP / (TP + FN)                         LBW class recall
 Specificity   = TN / (TN + FP)                         Normal class recall
 F1-Score      = 2×(Precision×Recall) / (P+R)           Imbalanced data
 G-Mean        = √(Sensitivity × Specificity)           Both classes fair
 AUC-ROC       = ∫ TPR d(FPR)                           Overall model rank
```

**Requirements:** R `>= 4.2`

```r
# ── STEP 1: Install all dependencies ──────────────────────────────────────────
install.packages(c(
  "tidyverse",      # data wrangling & visualization
  "caret",          # unified ML framework
  "Boruta",         # all-relevant feature selection
  "xgboost",        # gradient boosting (XGB)
  "randomForest",   # bagging ensemble (RF)
  "e1071",          # SVM kernel methods
  "shapley",        # SHAP explainability values
  "lime",           # local interpretable model-agnostic explanations
  "DMwR",           # SMOTE oversampling for class imbalance
  "pROC",           # ROC curve analysis & AUC
  "ROCR"            # performance visualization curves
))

# ── STEP 2: Run the full pipeline ─────────────────────────────────────────────
source("LBW_Prediction.R")
```
